日本 GlassFish ユーザー会

Community for Jakarta EE Developers and Users in Japan

日本 GlassFish ユーザー会 (GlassFish Users Group Japan) は、Jakarta EE の開発者およびユーザーが集まり、Jakarta EE および関連技術に関する情報を日本語で共有するための Java ユーザー・グループ (JUG) です。

Recent Activities

  • 2018-09-18 開催

    【JDK 11 リリース直前】各ベンダーのJDKリリースモデル特集！

    今回の勉強会では、主要JDKベンダーの中からAzul・IBMの2社をお招きし、今後のJDKリリースモデルや各社のLTS、サーバーでのサポートなどについて存分に語っていただきます。

  • 2018-04-27 開催

    GlassFish Users Group Japan 勉強会 2018 Spring

    今回はゲストスピーカーとしてSOMPOシステムズの西野大介様をお招きし、IBM - Think 2018 で Ian Robinson をはじめとする世界中の Java スペシャリストから得た生の声を、お伺いしようと思います。

  • 2017-12-19 開催

    GlassFish Users Group Japan 勉強会 2017 Winter

    2017 年最後の GlassFish Users Group Japan 勉強会 (兼 忘年会) を開催します。今回は株式会社カサレアル様の会議室をお借りして、いつもより小さくても熱く、Java EE と GlassFish/Payara に触れる一夜にしたいと思います。

  • 2017-06-06 開催

    Payara初来日記念！「Payara Meetup」

    この度、Payara の Steve Millidge 氏と Dominika Tasarz-Sochacka 氏が来日されるのに合わせて、Payara と日本 GlassFish ユーザー会が共同でイベントを開催することになりました。欧州を拠点とする Payara は、今回が実は初来日となります！記念すべきこの日をみんなで大いに楽しみましょう。
