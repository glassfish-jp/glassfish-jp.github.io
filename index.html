GlassFish Users Group Japan

Community for Jakarta EE Developers in Japan

日本 GlassFish ユーザー会 (GlassFish Users Group Japan) は、日本の Jakarta EE 開発者・ユーザーが集まり、互いに技術を研鑽し親睦を深めあうことを目的とした Java ユーザー・グループ (JUG) です。Java Community Process のメンバーであり、日本では数少ない、特定の技術に特化した JUG です。

日本 GlassFish ユーザー会の主役は、会員ひとりひとりです。それぞれの役割こそ異なりますが、会員が力を合わせて全員で作り上げていくコミュニティーを目指しています。

    GlassFish Users Group Japan 勉強会 2017 Winter

    2017 年最後の GlassFish Users Group Japan 勉強会 (兼 忘年会) を開催します。今回は株式会社カサレアル様の会議室をお借りして、いつもより小さくても熱く、Java EE と GlassFish/Payara に触れる一夜にしたいと思います。

  • 2017-06-06 開催

    この度、Payara の Steve Millidge 氏と Dominika Tasarz-Sochacka 氏が来日されるのに合わせて、Payara と日本 GlassFish ユーザー会が共同でイベントを開催することになりました。欧州を拠点とする Payara は、今回が実は初来日となります！記念すべきこの日をみんなで大いに楽しみましょう。
