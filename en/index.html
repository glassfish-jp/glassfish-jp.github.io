GlassFish Users Group Japan

Community for Jakarta EE Developers in Japan

GlassFish Users Group Japan is a Java User Group (JUG) that is consisted by Jakarta EE developers and users in Japan. We are a member of Java Community Process and specify for Jakarta EE and relative technologies.

Everyone are main actor in the group however each roles are different. We all create the future of the community.
